# EOG-Driven Robotic Manipulator control 

## Project Overview
This repository contains the source code, hardware designs, and documentation for an Electrooculography (EOG) control interface. The **ultimate objective** of this project is to develop a fully EOG-driven robotic control system, allowing users to navigate or command robotic hardware using eye movements alone. 

As a primary milestone toward full robotic integration, the current system successfully captures biopotential signals generated by eye movements to dynamically control a multi-position servo motor. This serves as the foundational steering and actuation framework for the final robot.

---

## Project Milestones & Current Evolution
* **Signal Capture & Mapping:** Successfully mapped horizontal/vertical eye movements to servo motor angles.
* **Hardware Limitation Identification:** Evaluated the performance of the native microcontroller ADC.
* **High-Resolution Upgrade (Current Phase):** Integrating an external high-precision ADC to clean up the signal before scaling up to full robotic control.
* **Full Robotic Integration (Final Goal):** Translating the refined EOG signals into complex multi-axis robotic movements.

## Hardware Evolution & Architecture

### The Challenge with the Onboard 10-bit ADC
Initially, the project used the built-in **10-bit ADC** of the Arduino. Testing revealed key bottlenecks:
* A 10-bit resolution provides only about 4.88mV per step.
* Because raw EOG signals sit in the microvolt to low millivolt range, the onboard ADC lacked the granularity required to distinguish subtle eye movements from baseline noise, resulting in jittery servo movement.

### The Solution: Upgrading to 16-bit Precision
To achieve the stability required for reliable robotic control, the system architecture is being upgraded to integrate the **ADS1115**, an external **4-channel, 16-bit Delta-Sigma ADC**.

* **Onboard Arduino ADC:** 10-bit resolution (1,024 steps), Internal Bus, No Programmable Gain, High Quantization Noise.
* **ADS1115 External ADC:** 16-bit resolution (65,536 steps), I2C Interface, Built-in PGA for weak biological signals, Stable Noise-Free Robotic Navigation.

By utilizing the ADS1115's built-in Programmable Gain Amplifier (PGA) and massive jump in quantization resolution, the system isolates genuine ocular tracking data from environmental noise—providing the smooth, reliable data stream necessary to safely control a robot.
